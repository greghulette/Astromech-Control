#include <core/SetupEvent.h>

Detailed Description

Utility class to control a VMusic2 module.

Base class for all devices that require setup that cannot happen in the constructor.

Provides for sound playback as well as disk access.

SetupEvent::setup() is called for each device once from the sketch setup() routine.
